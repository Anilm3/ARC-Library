Detailed Description

Generic iterator.

Author:
Anil M. Mahtani Mirchandani
Date:
June, 2014

A generic iterator is a common type of iterator that can be used for any data structure. It is not allowed to use the same iterator for other data structures other than the one specified for the iterator

For more information and examples check the documentation (page_iterator).

See also:
http://en.wikipedia.org/wiki/Iterator
