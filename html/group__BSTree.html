Detailed Description

Binary Search Tree.

Author:
Anil M. Mahtani Mirchandani
Date:
December, 2014

A binary search tree is a data structure in which each node contains a right and a left link. The right link points to a node containing a value bigger than the current nodes value, similarly the left link points to a node containing a value smaller than the current nodes value, hence the data structure is inherently ordered.

For more information and examples check the documentation (BSTree).

See also:
http://en.wikipedia.org/wiki/Binary_search_tree
