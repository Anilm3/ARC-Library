Detailed Description

Singly-linked list.

Author:
Anil M. Mahtani Mirchandani
Date:
June, 2014

A singly linked list is a structure in which the data is linked together in one direction only. The implementation of the structure is highly based on iterators in order to provide mechanisms to access all the elements in the structure.

For more information and examples check the documentation (SList).

See also:
http://en.wikipedia.org/wiki/Linked_list#Singly_linked_list

Typedef Documentation

arc_slist_t

List definition.

The list is defined as a pointer to be used with the creation and destruction functions, direct stack allocations are not allowed.
